# Standard Library: Floating-Point Math (std:fmath)

The `std:fmath` module provides mathematical functions for floating-point numbers. Import it with:

```seq
use std:fmath
```

## Trigonometric Functions

```seq
3.14159 fmath.sin    # sine
3.14159 fmath.cos    # cosine
1.0 fmath.tan        # tangent
```

## Inverse Trigonometric

```seq
0.5 fmath.asin       # arc sine
0.5 fmath.acos       # arc cosine
1.0 fmath.atan       # arc tangent
```

## Exponential and Logarithmic

```seq
2.0 fmath.exp        # e^x
10.0 fmath.ln        # natural log
100.0 fmath.log10    # log base 10
```

## Power and Roots

```seq
2.0 3.0 fmath.pow    # 2^3 = 8.0
16.0 fmath.sqrt      # square root = 4.0
27.0 fmath.cbrt      # cube root = 3.0
```

## Other Functions

```seq
-5.5 fmath.abs       # absolute value = 5.5
3.7 fmath.floor      # floor = 3.0
3.2 fmath.ceil       # ceiling = 4.0
3.5 fmath.round      # round = 4.0
```

## Constants

```seq
fmath.pi             # 3.14159...
fmath.e              # 2.71828...
```
